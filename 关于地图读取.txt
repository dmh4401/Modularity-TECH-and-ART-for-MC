目前已知可读取leveldb的python库
pylru：pylru是一个Python的LRU（最近最少使用）缓存库，虽然不是官方的LevelDB库，但可以方便地在Python中使用LRU缓存来提高查询性能。
安装方式：使用pip命令安装pylru库：pip install pylru
plyvel：plyvel是一个Python的LevelDB库，它提供了对LevelDB数据库的完整访问功能，并支持高级特性如迭代器、批量写入等。
安装方式：使用pip命令安装plyvel库：pip install plyvel
请注意，plyvel依赖于LevelDB C++库，因此在安装之前需要先安装LevelDB C++库。具体安装方法可参考官方文档
leveldb：leveldb是Google官方提供的Python库，用于直接操作LevelDB数据库。
安装方式：使用pip命令安装leveldb库：pip install leveldb
以上是几个常用的LevelDB库，根据自己的需求选择适合的库来操作LevelDB数据库。每个库都有自己的特点和用法，可以根据具体情况选择使用。
