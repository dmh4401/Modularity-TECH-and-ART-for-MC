指令链（CBC）用于储存连续的一串命令方块，使用链表结构
-------------------------------------------
header 储存该指令链主要信息
format_version 模板版本
minecraft_version 该指令链使用游戏版本
minecraft_edition 该指令链使用游戏类型

modules 储存指令链内容
data 储存指令链一个节点的数据域
name  命令方块名称
command 这里输入指令内容
type 命令方块类型 "p"为脉冲，"r"为重复，"c"为链
tickdelay 命令方块执行延迟时间刻，默认应为0
redstonemode 是否需要使用红石控制 布尔
conditionalmode 是否有条件约束 布尔
rotation 命令方块相对前一个命令方块的旋转 "s"为与上个命令方块同方向straight，"r"为相对于上个命令方块右转right，"l"为相对于上个命令方块左转

p 指针域
head头指针

comment 注释
-----------------------------------------------
3维的指令链被制作的几率极小，不在考虑范围内
第一个命令方块的方向应为程序内指定，其余命令方块在第一个命令方块方向的基础上衍生
在工程文件中储存时应当储存每个命令方块的坐标和方向，并标注记录首命令方块位置便于指令链判定
